
# Financial Market Behaviour Analysis

## Overview

This project analyses historical financial data from the S&P 500 and major technology stocks (Apple, Microsoft, Tesla) to explore patterns in returns, volatility, and correlation.

The goal is to understand whether financial markets behave consistently over time or exhibit changing conditions.

---

## Key Features

* Multi-asset financial data analysis
* Daily return calculation
* Volatility and rolling volatility analysis
* Correlation analysis across assets

---

## Key Findings

* Volatility is not constant and shows clustering behaviour
* Different assets exhibit significantly different risk profiles
* Market conditions appear to shift between calmer and more volatile periods

---

## Tools Used

* Python
* pandas
* matplotlib
* yfinance

---

## Project Structure
